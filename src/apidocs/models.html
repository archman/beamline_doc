beamline.models module¶

This module is written for the purposes of elements modeling for accelerator:
1: manually define magnetic elements one by one and model the machine; 2: interpret lattice file (.lte file) to be modeled elements; 2: update (EPICS databases)/(EPICS PVs) with new configuration.

Author : Tong Zhang Created : 2016-03-18
